CONCEPTUAL DESIGN

Site Plan

This is a proposed site design for an existing dental practice. The design provides fencing to replace the existing six-foot chain-link fence that surrounds the site. The new fencing aims to be cost-effective by utilizing standard pressure-treated lumber, off-the-shelf hardware, and the existing steel vertical fence posts. Using these existing posts provides structural support and prevents the need to disrupt the concrete slab into which the vertical posts are anchored. The use of varying widths of lumber provides visual interest, and spacing of the lumber provides a measure of transparency. A custom-designed planter is incorporated into each fence panel at varying heights.
